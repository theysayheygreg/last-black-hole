# LBH Two-Branch Sonic Exploration

**Prompt:** parallelize two possible LBH score identities under the same constraints:
- lo-fi / chiptune / SNES-limited voice budget
- procedural / system-aware implementation viability
- still must answer LBH's ASCII-first visual identity

---

## Branch A — Horror Machine Cathedral

**Anchor:** Caves of Qud – "Lazarus (B Side)"

### Core fantasy
This world has been running longer than memory. The machinery predates the player, the cult, maybe even the current cosmology. The high synth line is not "melody" so much as **ritualized devotion around incomprehensible machinery**.

### Emotional read
- ancient
- reverent
- low-end oppressive
- patient
- inhumanly continuous
- the player enters a process, not a place

### LBH translation
This branch is strongest for:
- wells as worshiped machines
- portal spaces as forbidden sanctums
- inhibitor as doctrinal rupture
- collapse as liturgical failure

### System mapping
- **World floor:** sub-heavy machine prayer / generator bed
- **Wells:** low sine/saw columns with ritual upper partials
- **High octave line:** cult-organ devotion above the abyss
- **Loot / reward:** not cheerful — more like "forbidden knowledge acknowledged"
- **Extraction:** temporary absolution, not triumph
- **Death:** the machine continues without you

### Audio rules
- low-end anchors the branch
- upper partials are sparse and ceremonial
- rhythmic content should be slow / processional, not groove-led
- use limited voices, but long-held relationships
- less horizontal movement, more vertical weight

### Risks
- can become too close to Metroid/Qud cave-religion ambience
- may suppress gameplay readability if the bed is too monolithic
- can become sonically magnificent but too static

---

## Branch B — Psychedelic Drift Engine

**Anchor:** Weedpecker – "Nothingness"

### Core fantasy
The universe is absurdly happening in real time, and the soundtrack does not deny it. This branch treats LBH as a **systems groove**: motion, drift, risk, extraction, and collapse all become rhythm-bearing phenomena.

### Emotional read
- fatalistic momentum
- shrugging cosmic intoxication
- propulsion over reverence
- music-forward
- body-moving systems awareness
- "well this is happening"

### LBH translation
This branch is strongest for:
- thrust / surfing / flow-state play
- event sync with system pulses
- more music-forward runs
- signature-specific groove identities
- eventual Crypt of the NecroDancer-style system-to-beat mapping

### System mapping
- **World floor:** fuzzed low groove + dusted arpeggio or ostinato fragments
- **Wells:** modal bass gravity and detuned drone pressure
- **Portals:** brighter hook fragments / harmonic openings
- **Scavengers:** rhythmic interruptions or hostile syncopations
- **Pulse / merge:** beat-scramble and compression events
- **Extraction:** stronger cue identity and propulsion

### Audio rules
- preserve low voice count
- groove must remain legible under system changes
- use SNES/chiptune constraints as contour, not nostalgia cosplay
- beats should arise from systems, not from a detached backing track

### Risks
- can overpower the game if too song-forward
- can feel tonally wrong if it becomes cool-rock swagger instead of cosmic danger
- easy to lose loneliness and dread

---

## Shared Constraint: SNES / lo-fi / limited voices

Neither branch should become high-fidelity excess.
The right shared discipline:
- 4–6 meaningful voices at once
- dirty low-pass ceiling
- bit-depth abrasion
- focused role separation
- no lush cinematic spread
- all beauty must survive scarcity

---

## Best use inside LBH

### If choosing Branch A as primary
Use Branch B as a secondary layer for:
- late-run propulsion
- extraction acceleration
- specific signatures like `the rush`

### If choosing Branch B as primary
Use Branch A as the metaphysical underbed for:
- wells
- collapse
- inhibitor
- title/menu

---

## My current read
LBH probably should not choose only one.
The strongest score may be:
- **Branch A for ontology** — what the universe is
- **Branch B for agency** — what the player is doing inside it

In plain language:
- the universe is ancient, cultic, machine-haunted
- the run itself develops groove, propulsion, and risky momentum

That hybrid would be much more LBH-specific than either branch alone.
